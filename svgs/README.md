# SVG Animations Project

A demonstration of SVG graphics and CSS animations without JavaScript.

## 📌 Project Overview
This project focuses on animating SVG elements using CSS keyframes.

## ✨ Features
- Bouncing circle animation
- Rotating square animation
- Line drawing animation using stroke-dashoffset
- Infinite looping animations

## 🛠 Technologies Used
- HTML5
- CSS3 (Keyframes, Transforms)
- SVG

## 📚 Concepts Practiced
- SVG element structure
- CSS transform properties
- Keyframe animations
- Stroke animation techniques

## 🚀 How to Run
Open `index.html` in your browser.

## 🎯 Learning Outcome
Gained understanding of vector graphics and animation principles.
