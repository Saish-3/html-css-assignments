# Surfing Website

A beach-themed responsive landing page showcasing surfing culture and locations.

## 📌 Project Overview
Built to practice background image handling and card-based section layout.

## ✨ Features
- Full-screen hero section
- Surf spots cards
- About section
- Contact section
- Responsive layout

## 🛠 Technologies Used
- HTML5
- CSS3 (Flexbox, Background Cover)

## 📚 Concepts Practiced
- Background image positioning
- Section-based layout design
- Responsive flexbox alignment

## 🚀 How to Run
Open `index.html` in your browser.

## 🎯 Learning Outcome
Improved skills in building visually engaging themed websites.
