# Contact Form Webpage

A clean and structured contact form built using HTML and CSS.

## 📌 Project Overview
This project focuses on designing and styling web forms with proper spacing, alignment, and input validation structure.

## ✨ Features
- Name input field
- Email input field
- Phone number field
- Message textarea
- Styled submit button
- Focus state styling
- Responsive layout

## 🛠 Technologies Used
- HTML5 (Form elements)
- CSS3

## 📚 Concepts Practiced
- Form structure and input types
- Input validation using HTML attributes
- Styling input focus states
- Layout alignment
- Box shadows and spacing

## 🚀 How to Run
Open `index.html` in your browser.

## 🎯 Learning Outcome
Developed understanding of form design and user input structure.
