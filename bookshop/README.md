# Bookshop Website

A responsive bookstore website showcasing books using a card layout design.

## 📌 Project Overview
This project demonstrates card-based UI structure and clean typography layout.

## ✨ Features
- Hero section
- Book cards with pricing
- About section
- Contact section
- Responsive design

## 🛠 Technologies Used
- HTML5
- CSS3 (Flexbox, Card Layout)

## 📚 Concepts Practiced
- Card UI structure
- Typography styling
- Responsive spacing
- Section organization

## 🚀 How to Run
Open `index.html` in your browser.

## 🎯 Learning Outcome
Strengthened understanding of UI layout consistency and structured design.
