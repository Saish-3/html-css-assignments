# Coffee Shop Landing Page

A modern coffee shop themed landing page built using HTML and CSS.

## 📌 Project Overview
This project demonstrates how to design a visually appealing themed website using background images, styled sections, and structured layout.

## ✨ Features
- Full-screen hero section
- Menu section with pricing
- About section
- Contact section
- Themed color palette
- Responsive layout

## 🛠 Technologies Used
- HTML5
- CSS3 (Flexbox, Background styling)

## 📚 Concepts Practiced
- Background image usage
- Section spacing and alignment
- Button hover effects
- Card-based layout
- Responsive design using media queries

## 🚀 How to Run
Open `index.html` in your browser.

## 🎯 Learning Outcome
Learned how to design themed landing pages and improve UI consistency.
