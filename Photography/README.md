# Photographer Portfolio Website

A responsive photography portfolio website showcasing image gallery layout using CSS Grid.

## 📌 Project Overview
This project demonstrates how to create an image-focused website layout suitable for photographers.

## ✨ Features
- Hero section with background image
- Responsive image gallery grid
- Hover zoom effect on images
- About section
- Contact section

## 🛠 Technologies Used
- HTML5
- CSS3 (CSS Grid, Transitions)

## 📚 Concepts Practiced
- CSS Grid layout
- Image object-fit property
- Hover animations
- Responsive image scaling

## 🚀 How to Run
Open `index.html` in your browser.

## 🎯 Learning Outcome
Improved understanding of grid layouts and image presentation techniques.
